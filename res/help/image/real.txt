Written as: `|real <tag_search>`
`<tag_search>` is a single realbooru tag to search by. Multiple tags must be concatenated with `+` (no spaces).
Tags themselves can include spaces, underscores (_) are added automatically.

Valid Examples:
:white_small_square: `|real <tag>`
:white_small_square: `|real <tag1>+<tag2>`

Invalid Examples:
:white_small_square: `|real`
:white_small_square: `|real <tag1>, <tag2>`
