**Nekos Life Commands**
These can include a username as an action to the specified member
```yml
- 8ball/ball/fortune
- cuddle
- hug
- kiss/mwah/smooch
- pat/pet/pap
- slap/smack
- smug
- tickle
```
In addition is the `neko/catgirl` command, which can include a `gif` argument. As well as `kemono/animalgirl` and `kitsune/foxgirl` (Image only)
**Examples**
```yml
- neko
- catgirl gif
```
**Nekos Fun Commands**
The nekos.life API removed the NSFW endpoints. The new NSFW commands utilize the nekos.fun API
