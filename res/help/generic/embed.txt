Written as: `<prefix>embed <message>`
`<message>` is a message the user wishes to be embedded
Optionally can include a hexidecimal number representing a colour by including `--#<colour_hex>`

Valid Examples:
:white_small_square: `<prefix>embed This is a message`
:white_small_square: `<prefix>embed --#ff0000 This is a colour`
:white_small_square: `<prefix>embed colour goes --#ae5a2f here`

Invalid Examples:
:white_small_square: `<prefix>embed`
:white_small_square: `<prefix>embed invalid colour --#00ffaaf`
